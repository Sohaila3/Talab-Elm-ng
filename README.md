# Talab Elm - طلب علم

مشروع Angular لإدارة الدورات التدريبية والشهادات المهنية.

## التثبيت

```bash
cd ~/Desktop/talab-elm
npm install
```

## تشغيل المشروع

```bash
npm start
```

ثم افتح المتصفح على: http://localhost:4200

## بناء المشروع للإنتاج

```bash
npm run build
```

## المميزات

- واجهة عربية RTL
- تصميم حديث وجذاب
- إدارة الدورات والشهادات
- فلترة الدورات حسب النوع
- تصميم متجاوب للشاشات المختلفة

## التقنيات المستخدمة

- Angular 17
- TypeScript
- CSS3
- Google Fonts (Cairo)
